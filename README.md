# 礼物弹窗温馨提示应用

这是一个基于 Vue 2 开发的互动式礼物弹窗应用，通过精美的 UI 设计和流畅的动画效果，为用户带来惊喜和温馨的体验。

## 项目功能

### 🎁 礼物确认流程

- 初始礼物确认弹窗，提供"查收"和"取消"选项
- 二次确认机制，增加互动趣味性
- 取消操作时随机更换温馨提示文字

### 💌 温馨提示弹窗

- 点击确认后自动生成大量精美小弹窗（最多 200 个）
- 6 种不同主题颜色，视觉丰富多样
- 12 种不同的温馨提示文案，传递关怀
- 点击小弹窗触发抖动动画和视觉反馈

### 🎨 视觉设计

- 渐变背景和浮动效果，增强视觉吸引力
- 按钮悬停和点击动画，提升交互体验
- 平滑的过渡动画，增强用户体验
- 响应式设计，适配不同屏幕尺寸

## 技术栈

- **前端框架**: Vue 2.6.14
- **样式**: CSS3 (渐变、动画、变换)
- **构建方式**: 单文件 HTML 应用（无需构建工具）

## 快速开始

### 安装和运行

1. 克隆或下载项目文件
2. 直接在浏览器中打开 `🐏.html` 文件
3. 无需额外配置，即可体验完整功能

## 项目结构

```
├── 🐏.html  # 主应用文件
```

## 使用说明

1. **礼物确认**: 页面加载后会显示礼物确认弹窗
2. **交互操作**:
   - 点击"查收"按钮，开始展示温馨提示弹窗
   - 点击"取消"按钮，进入二次确认
   - 再次点击"取消"，会更换提示文字
3. **弹窗互动**: 点击任意温馨提示弹窗，享受抖动动画反馈

## 自定义配置

您可以在 Vue 实例的`data`对象中轻松自定义以下内容：

- **提示文案**: 在`messages`数组中添加或修改温馨提示内容
- **弹窗主题**: 在`themes`数组中添加或修改颜色主题
- **二次确认文案**: 在`secondaryMessages`数组中自定义
- **最大弹窗数量**: 修改`MAX_NOTES`常量
- **初始弹窗数量**: 修改`initialNotesCount`常量
